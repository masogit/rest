Setup REST Agent

  1. Deploy rest-gateway in you UCMDB server Download package
  2. Edit config.properties file in package
  3. Run startup.bat in package (latest Java is required)
  4. Configure below parameters in Dashborad UI, browser will save your settings for next time:
Sever
Input your rest-gateway URL and PORT (Sample: 16.165.219.18:9091)
View
Input your TQL name
T Num.
How many Timeline number will be kept
P Size
Page size of pagination
Refresh
Interval to query server (second)

Features

  • Statistics will display all return CI number by CI Type
  • Click each CI Type, data will display in Records section
  • Required to define Element Layout for each CI Type

Setup Your Own Dashboard Server (not required)

  1. Install Node.js in your OS
  2. Download dashboard package dashboard.zip
  3. Unzip package and edit server.js before start serivce
  4. To start service, run in command line: node server.js
