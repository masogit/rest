UCMDB REST Client
Setup REST Agent
  1. Deploy rest-gateway in you UCMDB server Download package
  2. Edit config.properties file in package (UCMDB Info)
  3. Edit gateway.home(this package installed folder) in startup.bat (java -Drest.gateway.home=C:\hp\UCMDB\REST -jar rest-gateway-1.0.jar)
  4. Run startup.bat in package (latest Java is required)
Sever
Input your rest-gateway URL and PORT (Sample: 16.165.219.18:9091)
View
Manually input TQL view name, after query this view name will be saved in history
History
Display all views your query before, click button X to remove items
Customize
Modify your TQL, for example: update filter; add Element Layout etc
Install this Web APP
Git
Install Git, then git clone https://github.com/masogit/rest.git
NodeJS
Install NodeJS, then npm install to install components
Specify Port
Windows:set PORT=80 Linux:export PORT=80 default is 8080
Start Service
node server.js

Browser will save your last parameters automatically.
